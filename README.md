202155183 지민근 역할 

2017년부터 2019년까지 공정거래위원회가 발표한 정보를 기반으로 한 자료를 크롤링하고 출력하는 프로그램을 코딩하고 이를 git에 업로드 한다.



202155154 안정빈 역할
